# Changesets

Folder ini dikelola oleh [@changesets/cli](https://github.com/changesets/changesets).

## Cara Menggunakan

### 1. Buat changeset saat melakukan perubahan:
```bash
pnpm changeset
```

### 2. Bump versi berdasarkan changesets:
```bash
pnpm version-packages
```

### 3. Publish ke npm:
```bash
pnpm release
```
